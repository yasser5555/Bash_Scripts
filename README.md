# Bash Scripts Collection

A collection of useful Bash and Batch scripts for automating daily tasks, improving productivity, and simplifying command-line workflows.

## 📌 About

This repository contains different scripts written in **Bash** and **Batch (.bat)** to help automate repetitive tasks on Linux and Windows systems.

The goal of this repository is to:

* Practice shell scripting
* Automate common tasks
* Build reusable command-line utilities
* Learn scripting concepts through real examples

Bash is commonly used for automation and system administration in Linux environments. ([GitHub][1])

# ⚡ Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/yasser5555/Bash_Scripts.git
```

## 2️⃣ Navigate to the Folder

```bash
cd Bash_Scripts
```

## 3️⃣ Give Permission to Bash Scripts

```bash
chmod +x script.sh
```

## 4️⃣ Run the Script

### Linux / macOS

```bash
./script.sh
```

or

```bash
bash script.sh
```

### Windows Batch File

```cmd
script.bat
```
